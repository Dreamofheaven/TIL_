<시퀀스> 
-> string : char의 나열, "나 '로 표기
-> list :  변경가능하고 반복이 가능한 요소들이 콤마로 구분되어 나열 [로 정의
-> tuple : 변경이 불가능하고, 반복이 가능한 요소들이 콤마로 구분되어 나열, (로 정의
-> range : 변경이 불가능, for문에서 특정 횟수만큼 반복할 때 사용

<컬렉션>
-> set : 유일한 값들의 모음, 순서가 없으며 중복된 값이 없음 
-> dictionary : 키와 값이 쌍으로 이뤄진 모음 {로 정의
